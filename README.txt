# Sales Intelligence Hub

A Streamlit + MySQL based Sales Management Dashboard.

## Features

- Super Admin Login
- Admin Login
- Branch Wise Access Control
- Sales Entry
- Payment Entry
- Dashboard Analytics
- Payment Method Summary
- Customer Sales Reports
- SQL Query Analysis Module

## Technologies

- Python
- Streamlit
- MySQL
- Plotly

## Run Project

```bash
streamlit run dashboard_v2.py
```
